## About The Project

Command       |  Description
:-------------|:-------------
`make`        | Compile the library.
`make clean`  | Remove objects files.
`make fclean` | Remove objects files and the library.
`make re`     | Re-compile the library.
`make bonus`  | Compile the library with bonus.
`make help`   | Need some help ?
